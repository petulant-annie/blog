## Zazmic internship blog

Creating a blog backend as an internship in Zazmic.inc

Developed using:
 * Node.js, 
 * MySQL(Sequelize), 
 * MongoDB(Mongoose), 
 * Passport.js, 
 * Socket.IO, 
 * Redis, 
 * Google Cloud, 
 * SendGrid, 
 * Stripe

## Usage
```bash
npm i
npm start
```
